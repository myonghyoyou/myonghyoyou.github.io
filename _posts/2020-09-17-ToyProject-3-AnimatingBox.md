CSS 없이 순수 JavaScript 만으로 애니메이션에 대한 작업을 해보기 위해 작업한 AnimatingBox 프로젝트이다.

단편적인 Animation 작업은 CSS를 통해서 할 수 있지만 좀 더 세밀하고 자유도 있는 환경에서 Animation을 넣으려면 JavaScript에서 작업을 할 줄 알아야 한다는 글을 읽었다. \
추가적으로, 토이 프로젝트를 진행할 때마다 자신만만하게 시작하지만 되게 사소하다고 판단한 부분에서 막히는 스스로를 보며 기초가 아직 많이 부족하다는 생각이 들었다. \
기초와 관련된 부분을 갈고닦기 위해 한참 동안은 이와 같은 소소한 토이 프로젝트를 통해 HTML, CSS, JavaScript 모두 게속 연습할 예정이다. \

#### 학습 내용

  1. position을 static으로 둔 경우에는(default), z-index를 적용시킬 수 없다.
  2. JS의 animate 메소드를 사용하면 css의 keyframe을 사용하는 것과 같은 결과를 기대할 수 있다.
  3. transform-origin을 통해 transform의 기준점을 정해줄 수 있다. (default는 50% 50%)
  4. 막막해 보여도 하려고 하는 의지만 있으면 어떻게든 기능 구현은 할 수 있다.
