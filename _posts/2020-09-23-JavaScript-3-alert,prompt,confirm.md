### 개요

브라우저 환경에서 사용되는 최소한의 사용자 인터페이스 alert, prompt, confirm에 대한 학습을 진행한다.

---

### alert

`alert` 함수는 실행되면 사용자가 '확인(OK)' 버튼을 누를 때까지 메시지를 보여주는 창이 계속 떠있게 된다. 아래는 그 예시다.

```
alert("This is JavaScript");
```

메시지가 있는 작은 창은 모달 창(Modal Window) 이라고 부른다. '모달'이란 페이지의 나머지 부분과 상호작용할 수 없다는 의미를 내포하고 있다.
따라서 사용자는 모달 창 바깥에 있는 버튼을 누른다던가 하는 행동을 할 수 없다.

---

### prompt

`prompt` 함수는 두개의 인수를 받는다.

```
result = prompt(title, [default]);
```

함수가 실행되면 텍스트 메시지와 입력 필드, 확인 및 취소 버튼이 있는 모달 창을 띄워준다.

`title` = 사용자에게 보여줄 문자열

`default` = 입력 필드의 초깃값 (옵션)

>인수를 감싸는 대괄호?
>>해당 매개변수가 필수가 아닌 옵션이라는 뜻

사용자는 프롬프트의 입력 필드에 원하는 값을 입력하고 확인을 누를 수 있다. 값 입력을 원하지 않는 경우에는 취소 버튼을 누르거나 ESC 버튼을 누르면 된다.

값이 입력된 경우에는 입력된 값을 반환하며 사용자가 입력을 취소한 경우에는 `null` 값이 반환된다.

예시 :
```
let age = prompt("나이를 입력해 주세요", 20);

console.log("당신의 나이는 ${age}살 입니다.");
```

---

### confirm

```
let reuslt = confirm(question);
```

`confirm` 함수는 매개변수로 받은 question(질문)과 확인 및 취소 버튼이 있는 모달 창을 띄운다. 사용자가 확인을 누르면 true, 취소를 누르면 false를 반환한다.

예시 : 

```
let result = confirm("당신은 졸립니까?");

if(result == true){
  console.log("졸려죽겠네");
} else {
  console.log("멀쩡한 상태");
}
```

### 요약

브라우저는 사용자와 상호작용할 수 있는 3가지 함수를 제공한다.

`alert` \
메시지를 보여준다.

`prompt`\
텍스트를 입력하라는 메시지와 함께 입력 필드, 확인 및 취소 버튼을 보여준다. 메시지를 입력하고 '확인'을 누르면 입력된 값을 반환하며, 입력을 취소할 시 null을 반환한다.

`confirm`\
질문과 함께 확인 및 취소 버튼을 보여준다. 확인을 누르면 true, 취소를 누르면 false를 반환한다.

위 함수들은 모두 모달 창을 띄우며, 모달 창이 떠있는 동안에는 모든 스크립트의 실행이 중단된다. 뿐만 아니라, 모달 창에는 2가지 제약사항이 있다.

1. 모달창의 위치는 브라우저가 정하는데 대개 화면 중앙에 위치
2. 모달창의 디자인은 브라우저마다 달라지며, 사용자가 변경할 수 없다.
