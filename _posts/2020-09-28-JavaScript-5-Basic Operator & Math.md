### 개요

기본적인 연산자에 대한 개념은 있으나 자바스크립트에서만 제공하는 연산자에 대한 이해를 키우기 위해 학습을 진행한다.

---

### 용어 : '단항', '이항', '피연산자'

* 피연산자(operand) : 연산자가 연산을 수행하는 대상. `5 * 2` 에는 `5`와 `2` 이렇게 총 두 개의 피연산자가 있다. 피연산자는 인수라는 용어로 불리기도 한다.

* 단항(unary) : 피연산자를 하나만 받는 연산자를 '단항 연산자' 라고 부른다. 피연산자의 부호를 뒤집는 `-` 마이너스 연산자가 단항 연산자의 대표적인 예이다.

``` javascript
let x = 1;

x = -x;
alert(x); // -1, 단항 마이너스 연산자는 부호를 뒤집는다.
```

* 이항(binary) : 두 개의 피연산자를 받는 연산자는 '이항 연산자' 라고 부른다. 마이너스 연산자는 아래와 같이 이항 연산자로 쓸 수  있다.

``` javascript
let x = 1, y = 3;

alert(y-x); // 2, 이항 마이너스 연산자는 뺄셈을 한다.
```

피연산자의 개수에 따라 마이너스 연산자가 수행하는 기능이 달라진다. \
단항 마이너스 연산자는 부호를 반전하며, 이항 마이너스 연산자는 뺄셈에 쓰인다.

---

### 수학

자바스크립트에서 지원하는 수학 연산자는 다음과 같다.

`+`, `-`, `*`, `/`, `%`, `**`

#### 나머지 연산자 %

나머지 연산자를 사용한 표현식 `a % b`는 `a`를 `b`로 나눈 후 그 나머지를 정수로 반환한다.

예시 : 
``` javascript
alert( 5 % 2 ); // 1, 5를 2로 나눈 후의 나머지인 1 출력
alert( 9 % 7 ); // 2, 9를 7로 나눈 후의 나머지인 2 출력
```

#### 거듭제곱 연산자 **

거듭제곱 연산자를 사용한 `a ** b`는 `a`를 `b`번 곱한 값이 반환된다.

예시 : 
``` javascript
alert( 2 ** 2 ) // 4, ( 2 * 2 )
alert( 2 ** 4 ) // 16, ( 2 * 2 * 2 * 2 )
alert( 3 ** 3 ) // 27, ( 3 * 3 * 3 )
```

거듭 제곱 연산자는 정수가 아닌 숫자에 대해서도 동작한다. `1/2`를 사용하면 제곱근을 구할 수 있다.

예시:
```javascript
alert( 4 ** (1/2)) // 2, (1/2 거듭제곱은 제곱근)
alert( 8 ** (1/3)) // 3, (1/3 거듭제곱은 세제곱근)
```

### 이항 연산자 "+"와 문자열 연결

이항 연산자 `+`의 피연산자로 문자열이 전달되면 덧셈 연산자는 덧셈이 아닌 문자열을 연결한다. 피연산자 중 하나가 문자열이면 다른 하나도 문자열로 변환된다.

예시 : 
```
alert( 1 + '2'); // '12'
alert( 2 + '1'); // '21'
```

다른 산술 연산자가 오직 숫자형의 피연산자만 다루고, 피연산자가 숫자형이 아닌 경우에는 그 형을 숫자형으로 바꾸는 것과 반대로 이항 덧셈 연산자는 문자열 연결과 변환이 가능하다.

예시 : 

``` javascript
alert( 6 - '2' ); //4, '2'를 숫자로 바꾼 후 연산이 진행된다.
alert( '6' / '2' ); //3, '6'과 '2'를 숫자로 바꾼 후 연산이 진행된다.
```

### 단항 연산자 +와 숫자형으로의 변환

덧셈 연산자 `+`는 이항 연산자뿐만 아니라 단항 연산자로도 사용 가능하다.

피연산자가 숫자가 아닌 경우에 단항 덧셈 연산자를 붙이면 숫자형으로의 변환이 일어난다.

예시 :

``` javascript
// 숫자에는 아무런 영향을 미치지 않는다.
let x = 1;
alert ( +x ); // 1

// 숫자가 아닌 피연산자는 숫자형으로 변환한다.
alert ( +true ); // 1
alert ( +"" ); // 0
```

단항 덧셈 연산자는 `Number(value)` 와 동일한 일을 할 수 있게 해준다.

예시 :
``` javascript
let apples = "2";
let watermelon = "3";

alert( apples + watermelon ); // "23", 이항 덧셈 연산자는 문자열을 연결한다.

alert( +apples + +watermelon ); // 5, 이항 덧셈 연산자가 적용되기 전에, 두 피연산자는 숫자형으로 변환.

// alert( Number(apples) + Number(watermelon) ); 와 같이 작성할 수도 있지만 길어진다.
```

위처럼 이항 덧셈 연산자가 적용되기 전에 단항 덧셈 연산자가 적용되는 이유는 연산자 우선순위 때문이다.

---

### 연산자 우선순위

하나의 표현식에 둘 이상의 연산자가 있는 경우, 실행 순서는 연산자의 우선순위에 의해 결정된다.

자바스크립트에서 정의하고 있는 연산자가 마음에 들지 않을 경우에는, 괄호를 사용하면 된다. 괄호는 모든 연산자보다 우선순위가 높기 때문에 연산자 우선순위를 무력화시킨다.

동일한 기호의 단항 연산자는 이항 연산자보다 우선순위가 더 높다.
